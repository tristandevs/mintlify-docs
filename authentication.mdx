You may get a **FREE** ShuttleAI API key by performing the `/getkey` command in our [official discord server](https://discord.gg/shuttleai)!

All API requests should include your API key in an Authorization HTTP header as follows:

> Remember that your API key is a secret! Do not share your key with anyone!
```javascript ShuttleAI API Authorization
Authorization: Bearer SHUTTLEAI_API_KEY
```
