You can interact with the API through HTTP requests from any language, or from our [Official Python Library](https://pypi.org/project/shuttleai).

To get started with using the official Python library, read below:

## Installation

> We highly recommend using the async client instead of synchronous.

```sh
pip install -U shuttleai
```

from shuttleai import *

shuttle = Shuttleclient("ShuttleAI_API_KEY")

response = shuttle.chat_completion(
            model="gpt-3.5-turbo",
            messages=[{"role":"user","content":"write me a short story about bees"}],
            stream=False
         )
